# K-Means Clustering

A simple C implementation of K-Means clustering with bindings to Python and R.

## Get started

### Requirements

- GCC/Clang
- CMake 3.1+
- Python 3.6+ w/ numpy (optional)
- R 4.0+ (optional)

### Install directly

```sh
python -m pip install .
```

### Build wheel

```sh
python setup.py bdist_wheel
```
